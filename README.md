# syn
作者：	Ex
时间：	2018-08-02
更新时间：  2018-10-05
开发环境：	Linux mint
编译环境：  Centos 7 + gcc
测试运行环境：	Centos 7.*、Linux mint
编译选项：  -l pthread -std=c99 -O2
库依赖：	无特别库依赖

用法: syn [选项] -a [目标IP:目标端口]
  -h                                    显示帮助信息。
  -d                                    开启调试模式
  -a [目标IP:目标端口]                    你要测试攻击的目标。
  -i [源IP]                             设置数据包的源IP。
                                          默认为随机的源IP。
  -p [源端口]                            设置数据包的源端口。
                                          默认为随机的端口。
  -t [毫秒数]                            每次攻击后的延时（单位为毫秒）。
                                          默认为0。
  -f                                    开启极速模式，这需要设置 -i 和 -p 参数才能开启。
  -l [线程数目]                          设置线程数目。
                                          最大线程为128。
举例：./syn -a 192.168.3.1:80 -l 4

注：因为使用的是原始套接字，所以需要root权限运行

调试模式参数：
  h                                    显示帮助信息。
  q                                    退出程序。
  w                                    暂停所有线程。
  r                                    重启所有线程。
  s                                    展示所以信息。
  a [目标IP:目标端口]                    重新设置测试攻击目标。
  i [源IP]                             设置源IP
  p [源端口]                            设置源端口
                                          设置为0则为随机端口
  t [毫秒数]                            重新设置延时
